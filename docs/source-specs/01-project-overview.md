# Project Overview

## Scope

Application for two active wax flows:

1. Processing customer wax into wax foundations while wax remains custody stock.
2. Purchasing wax into company-owned stock and recording later stock exits.

## Principles

- Full operational traceability.
- Separate custody stock from company-owned purchased stock.
- Append-only movements and audit records.
- Store-scoped user operations.
- Editable document templates and PDF generation where documents are generated by the app.
- External paper/accounting documents are referenced, not generated, for current purchase flow.
