# Flow Procesare Ceară

## Statusuri
- In Validare
- Acceptat
- Predat Fabricii
- Respins
- Returnat

## Flux client cunoscut
1. PV Custodie
2. Acceptare
3. Factură
4. Bon fiscal
5. PV Predare Faguri
6. Status Acceptat

## Flux client necunoscut
1. PV Custodie
2. Status In Validare
3. Predare lot la fabrică
4. Acceptat sau Respins
5. Dacă acceptat -> Factură + PV Predare
6. Dacă respins -> PV Returnare
