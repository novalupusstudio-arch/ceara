# Users, Roles & Permissions

## Roles

### Admin

- manage users
- manage role permissions
- manage stores
- manage processors
- view audit

### Operator

- create processing lots
- create purchase lots
- work inside assigned store

## Recommended Permissions

- `USER_CREATE`
- `USER_EDIT`
- `USER_RESET_PASSWORD`
- `ROLE_PERMISSION_MANAGE`
- `STORE_MANAGE`
- `PROCESSOR_MANAGE`
- `PROCESSING_CREATE`
- `PROCESSING_ACCEPT`
- `PROCESSING_REJECT`
- `PURCHASE_CREATE`
- `REPORT_VIEW`
- `AUDIT_VIEW`
