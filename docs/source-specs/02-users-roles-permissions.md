# Users, Roles & Permissions

## Roluri
### Admin
- administrare utilizatori
- administrare roluri
- administrare gestiuni
- administrare procesatori
- administrare prețuri
- administrare scăzăminte
- administrare integrări
- acces audit

### Operator
- procesare ceară
- achiziție ceară
- generare documente
- acces doar la gestiunile alocate
